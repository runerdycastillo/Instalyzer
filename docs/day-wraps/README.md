# Day Wraps

Use this folder for dated end-of-day wrap-ups that collect the session summary and the milestone audit snapshot in one place.

Recommended structure:

- `docs/day-wraps/YYYY-MM-DD/end-of-day.md`
- `docs/day-wraps/YYYY-MM-DD/milestone-audit.md`

Keep `CURRENT_MILESTONE_AUDIT.md` as the live rolling document, then snapshot the relevant session state here at the end of the day.
